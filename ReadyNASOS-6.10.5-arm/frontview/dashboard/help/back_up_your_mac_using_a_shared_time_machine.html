Back Up Your Mac Using a Shared Time Machine

You can use your ReadyNAS server as the disk for Time Machine backups. ReadyNAS OS supports two different types of Time Machine targets, a single Time Machine shared by several users, and Time Machines private to an individual user.

Your ReadyNAS server can act as a shared Time Machine for multiple accounts, and can provide separate, or private, Time Machine disks for individual accounts. Use this procedure for a Private Time Machine.

Before performing these steps, verify your ReadyNAS server has the AFP protocol enabled. Note that it is enabled by default.

  1. Log into your ReadyNAS server.
  2. On the local admin page for your ReadyNAS, select Backup > Time Machine.

    User accounts already configured for a private Time Machine display here.

  3. Select the user, or click the plus (+) button to add the private Time Machine option to an existing user not already configured to use it.
