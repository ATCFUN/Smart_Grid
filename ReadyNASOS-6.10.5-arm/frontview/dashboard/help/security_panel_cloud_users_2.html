Cloud User Access Rights

When you grant access to ReadyCLOUD users, those users can access your ReadyNAS system using ReadyCLOUD. You use the ReadyCLOUD web portal to configure access rights for users accessing your system from ReadyCLOUD. See Manage Permissions for ReadyCLOUD Users.

If you did not enable anonymous access to a shared folder, anyone who tries to access system must provide valid ReadyNAS user account credentials.

For more information about managing access to shared folders on your system, see Set Network Access Rights to Shared Folders.
