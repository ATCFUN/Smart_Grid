Sync With Azure

You can sync files between your ReadyNAS system and your Microsoft Azure storage space.

You must configure Azure before setting up using with your ReadyNAS system. As part of the Azure setup process, you get both an Azure storage account and access key. You must enter both as part of the ReadyNAS synchronization setup. For information about Azure, visit https://portal.azure.com/.

To set up syncing:

  1. Log in to your ReadyNAS.
  2. Select Cloud.
  3. Set the Microsoft Azure On-Off slider so that the slider shows the On position.
    The Microsoft Azure window opens.
  4. Enter your Azure storage account and access key, select a value in the Region menu, and enter a name in the Blob Container Name field (or if you have already defined containers, select a name).
  5. Click the Next button.
  6. (Optional) Adjust the upload and download speed.
    For no limits, enter 0.
  7. Click the Next button.
    The Microsoft Azure - Sessions window opens.
  8. Enter a value in the Local Path field by browsing the ReadyNAS to the location that you want to sync.
  9. Enter a value in the Cloud Storage Path field by browsing the ReadyNAS to the location that you want to sync.
  10. In the Sync Direction menu, select Bidirection, Download cloud storage changes only, or Upload local storage changes only.
  11. Click the Create button.
    The Microsoft Azure - Session window closes.
The ReadyNAS applies the settings to sync with your Amazon S3 drive.
Note: You can add and modify the Azure drive session settings by clicking the Microsoft Azure Settings button.
