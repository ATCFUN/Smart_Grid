Access Shared Folders From a Network-Attached Device

You can remotely access shared folders and snapshots on your storage system using other network-attached devices, such as a laptop or tablet. The network-attached device must support one of the enabled file-sharing protocols. How a shared folder is accessed depends on the OS of the network-attached device, the file-sharing protocols that you enabled for shared folder access, and the access rights that you granted (see Shared Folder Access Rights).
Note: For snapshots to be accessible to users from their network-attached devices, you must select the Allow snapshot access check box on the shared folder settings window. For more information, see View and Change the Properties of a Shared Folder.
  1. Use a Web Browser
  2. Use a Windows Device
  3. Use a Mac OS X Device
  4. Use a Linux or Unix Device
  5. Use FTP and FTPS
  6. Use Rsync
