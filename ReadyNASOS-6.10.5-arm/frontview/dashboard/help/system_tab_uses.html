System Overview

You can use the System page to view basic information about your ReadyNAS system such as: the model, name, overall status, antivirus status, serial number, firmware version, device time, overview of the configured RAID groups, and about any installed applications. You can also change the name, antivirus settings, check for firmware updates, set the time, and change time settings.

  1. System Page Device Section
    Use the Device section to view such information as: the model, name, overall status, antivirus status, serial number, firmware version, device time, and overview of the configured RAID groups. You can also change the name, antivirus settings, check for firmware updates, set the time, and change time settings.
  2. System Page Apps Section
    Use the Apps section to view and launch installed apps.
  3. System Page Featured Apps Section
    Use the Featured Apps section to view a selection of apps highlighted by NETGEAR. You can install featured apps from this section.
