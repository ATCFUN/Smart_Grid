Configure SMB

To configure SMB:

  1. Log in to your ReadyNAS.
  2. Select System > Settings > Services.
    • If the indicator is green, the protocol is enabled.
    • If the indicator is gray, the protocol is disabled.
  3. Click the SMB protocol button.

    The SMB Settings window opens.

  4. Configure the settings as explained in the following table:
    Item Description
    Enable SMB Select the check box to enable SMB. Clear the check box to disable SMB.
    Enhance MacOS Select the check box to enable enhancements for use with MacOS. Clear the check box if you are not connecting to a MacOS system.
    Workgroup Default value is VOLUME, but you can change the workgroup name as needed.
    SMB3 Transport Encryption
    • Configure per share. Select the Configure per share radio button to enable SMB3 transport encryption for individual shares. To use encryption, encryption must be enabled on both the SMB client and SMB server.
    • Configure globally. Select the Configure globally radio button to require a single SMB3 transport encryption policy for all shares on all volumes. Use the menu to set one of the following policies:
      • Enabled
      • Disabled
      • Desired
      • Required
      To use encryption, encryption must be enabled on both the SMB client and SMB server.
  5. Click the Apply button.

    The SMB Settings window closes and the selected SMB configuration is set.

    If you set configure per share, turn encryption on or off for the share from the advanced share settings window (see Set Network Access Rights to Shared Folders).
