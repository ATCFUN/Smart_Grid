Delete User Accounts

This procedure assumes that your ReadyNAS system is currently in Local Users mode. For more information about changing user and group management modes, see User and Group Management Modes.

Use the local admin page to delete user accounts. Files on your ReadyNAS system that are owned by the deleted user might become inaccessible. When you delete a user, your ReadyNAS system deletes that userâ€™s private home folder and its contents.

To delete a user:

  1. Log in to your ReadyNAS.
  2. Select Accounts > Users.

    The list of current users displays.

  3. From the list of users, select the user account.
  4. Click the Delete button in the pop-up menu.
  5. Confirm the deletion.

    The user is deleted.
