Delete Groups

This procedure assumes that your ReadyNAS system is currently in Local Users mode. For more information about changing user and group management modes, see User and Group Management Modes.

Use the local admin page to delete a group. To be eligible for deletion, a group cannot contain any primary members. For more information about moving users to a different group, see Edit User Accounts. For more information about deleting users, see Delete User Accounts.

To delete a group:

  1. Log in to your ReadyNAS.
  2. Select Accounts > Groups.

    The list of groups displays.

  3. Select the group.
  4. Click the Delete button in the menu.
  5. Confirm the deletion.

    The group is deleted.
