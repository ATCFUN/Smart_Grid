Recovery to the Original Site

If you want the recovered share or LUN on the original ReadyNAS device, create a one time ReadyDR job to backup the ReadyDR share with the snapshots back to the original ReadyNAS device, and then clone the snapshots. For information on creating the ReadyDR job, see Create a ReadyDR Job. For information about cloning the snapshot, see Clone Snapshots.
