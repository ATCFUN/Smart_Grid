Enable File Search

You can search the files on your ReadyNAS system for file name, content, file type, file extension, size, and dates. If your ReadyNAS system is attached to a Mac, you can also use the Finder to search the files on the ReadyNAS.

To enable file searching:

  1. Log in to your ReadyNAS.
  2. Select System > Settings > Services.
  3. Click the File Search button.
  4. Select the Enabled check box.
  5. (Optional) Select the Content indexing check box.

    Content indexing adds indexing of several file types. The list of supported file types is available by clicking the question mark icon next to the Content indexing check box.

    Use the ADVANCED tab to specify types of files to exclude from indexing.

  6. Click the Apply button.

    The indicator on the File Search button turns green and file searching is enabled.
